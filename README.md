# TrainoPro - Modern Dark Tech Gym Dashboard

## 🎨 [CLICK HERE TO SEE THE DESIGN PREVIEW](https://focreatives-del.github.io/Xfitv2/)

### 🚀 About this Project
This is a premium gym dashboard integrated with the TrainoPro Workout Manager plugin.

### ✨ Features
- **Modern Dark Tech UI**: Glassmorphism, neon accents, and premium typography.
- **Persistent Metrics**: Water, nutrition, and health tracking saved to the database.
- **Workout Detail Pages**: Interactive exercise tracking for every muscle group.
- **Profile Overlay**: User registration and health goal management.

### 📂 Repository Structure
- `style.css`: The main design system.
- `front-page.php`: The dashboard logic.
- `plugin-files/`: The backend plugin code.
- `preview.html`: Static design preview for GitHub Pages.
